git init = initierar projektet till git, görs en gång per projekt

git add filenamn/foldernamn/ . // git stage (punkt betyder allt), antingen eller. Puschar projektet till git SPARAS LOKALT

git commit -m "en vettig kommentar för koll på vad som gjorts" SPARAS LOKALT

Dessa gör man flera gånger

git status - kolla status

git checkout . = hämtar alla filer man raderat

ls -a = kolla ifall git finns i ditt projekt
ls -la = se vilka filer som ligger i projektet

git remote remove origin = ta bort felkopplad
git remote -v = kolla versions

git log = visar vilka commits som har gjorts

clear - terminalen rensas
